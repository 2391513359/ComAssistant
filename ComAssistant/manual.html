串口调试助手帮助文件

文件

  • 保存原始数据：保存自上一次清空缓存后收到的任何原始数据
  • 读取原始数据：将文件中的数据读入接收缓存，暂时无用
  • 保存显示区数据：将数据显示区中的文本保存，所见即所得
  • 保存曲线数据：将绘图器收到的数据保存为excel文件
  • 保存曲线图片：将绘图曲线保存为图片。图片的尺寸由当前窗口大小决定。图像的质量顺序由高到底通常为pdf > bmp > jpg = png

功能

  • STM32-ISP：实现STM32F1和F4系列的芯片的一键下载功能。该功能同时依赖自动下载电路，否则应首先使单片机进入ISP模式
  • 多字符串：发送的数据均会记录下来供重复使用。

设置

  • 串口设置：对串口进行更丰富的设置
  • 编码规则：选择软件内部使用的编码规则。这会影响中文显示效果
  • 回车样式：选择win风格回车"\r\n"还是unix风格回车"\n"。这会影响回车效果
  • 关键字高亮：开启/关闭数据显示区的关键字高亮功能，方便阅读。时间戳信息、以及如下字符, ; { } : = ( ) [ ] < >
  • 恢复默认设置：软件支持保存部分设置，该按钮可以恢复所有默认设置。也可以在关闭软件后通过删除程序目录的config.ini文件达到同样目的。

绘图器

  • 开启：绘图器开关
  • 协议类型：绘图器支持两种简单的协议类型
    • Asic: 协议规则为：printf("{:%f,%f,%f}", data1, data2, data3);。变量数量不能超过15个
    • Float: 协议规则为直接将float类型数据按小段顺序拆成4个字节依次发送，并以0x00, 0x00, 0x80, 0x7F四个字节作为一组数据的结尾。变量数量不能超过15个
  • 线型：支持线图、点线图、点图三种风格

操作

  • 选中曲线时可以：右键删除、隐藏选中曲线，以及坐标跟随
  • 右键图例可以修改图例显示位置
  • 选中轴时可以：对选中的轴进行缩放
  • 双击曲线名称可以修改曲线名称
  • 双击轴标题可以修改轴标题名称
  • “数据显示区”、“多字符串发送区”、“绘图区”之间均可以通过拖动自行调整大小
